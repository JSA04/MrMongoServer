Collections

O MongoDB armazena documentos em coleções. Coleções são análogas às tabelas do banco de dados relacional. No entanto, diferentemente das tabelas, os documentos em uma coleção não precisam seguir um esquema fixo, permitindo que diferentes documentos dentro da mesma coleção possam ter campos diferentes e estruturas variadas. Isso proporciona uma flexibilidade significativa no armazenamento e na modelagem de dados, adequando-se bem a aplicações com requisitos dinâmicos e dados semiestruturados.

Como criar um collection?

Assim como as databases, o MongoDB não possui um comando para criar uma collection. No entanto a forma de criar uma collection é inserindo um documento com o algum método de inserção, e assim, especificando o nome da collection.
