Databases

Um database (ou banco de dados) é uma coleção organizada de informações ou dados que são armazenados e acessados eletronicamente. No caso do MongoDB, os dados são armazenados em documentos BSON (Binary JSON), que permitem uma representação flexível e dinâmica dos dados.

Como criar um database?

O MongoDB não possui um comando para criar um database, mas sim para usar um database. E quando um database não existe basta usar este comando que assim que o primeiro registro for inserido no banco, ele será criado.

use nomeDoBanco;
