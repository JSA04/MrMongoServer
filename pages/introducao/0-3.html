Documentos

O MongoDB armazena registros de dados como documentos BSON. BSON é uma representação binária de documentos JSON, embora contenha mais tipos de dados do que JSON.

Sintaxe

{ _id: ObjectId("603k1f77bcaa6cd799fg901a") nome: "Edops Toledo", saldo: 1780.39, data_nascimento: ISODate("2008-02-20T20:14:14.796Z"), ativo: true, categorias: ["Jogos", "Futebol"], endereco: { logradouro: "Rua José Bicudo", numero: 123, bairro: "Osasco", cidade: "São Paulo", estado: "SP" } }

Tipos de BSON

Tipo Descrição Exemplo
ObjectId Um identificador único para documentos. ObjectId("603k1f77bcaa6cd799fg901a")
String Uma sequência de caracteres de texto. "Edops Toledo"
Double Número de ponto flutuante de 64 bits. 1780.39
ISODate Data em formato ISO-8601. ISODate("2008-02-20T20:14:14.796Z")
Boolean Valor booleano (true ou false). true
Array Uma lista de valores. ["Jogos", "Futebol"]
Document Um documento aninhado (subdocumento). {"logradouro": "Rua José Bicudo", "numero": 123, "bairro": "Osasco", "cidade": "São Paulo", "estado": "SP"}
