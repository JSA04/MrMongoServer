insertMany()

As operações de criação ou inserção adicionam novos documentos a uma coleção. Se a coleção não existir no momento, as operações de inserção criarão a coleção. Os comandos para inserção são divididos em dois sendo eles o insertOne() e o insertMany().

O comando db.collection.insertMany() recebe um array de documentos e insere todos de uma vez na collection especificada.

S

Sintaxe

db.jogadores.insertMany( [ { nome: "Jonathan Palleri", idade: 30, time: "Santo Paulo" }, { nome: "Angel Pomero", idade: 31, time: "Cortinas" }, { nome: "Poão Paulo", idade: 28, time: "Prantos" }, { nome: "Rafael Peiga", idade: 28, time: "Parmeiras" } ]);
