Exercícios

Contexto do Banco

Temos um banco chamado dbAcademia contendo uma coleção de alunos com os seguintes campos:

alunos{ id: 1, nome: "Carlos Silva", idade: 25, plano: "Mensal", atividades: ["Musculação", "Crossfit"], mensalidadePaga: true }

Exercícios com DeleteOne

Exercício 01

Remova um aluno específico pelo nome. Remova o aluno cujo nome é "Carlos Silva".

dbAcademia.alunos.( {nome: } );

Exercicio 02

Remova um aluno pelo ID. Remova o aluno com id 1.

dbAcademia.alunos.( {: } );

Exercícios com DeleteMany

Exercício 01

Remova todos os alunos com plano mensal. Remova todos os alunos que possuem o plano "Mensal".

dbAcademia.alunos.( {: } );

Exercicio 02

Remova todos os alunos que não pagaram a mensalidade. Remova todos os alunos cujo campo mensalidadePaga é false.

dbAcademia.alunos.( {: } );
