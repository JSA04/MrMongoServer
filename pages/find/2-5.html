Operadores de Matrizes

Os operadores de matrizes permitem realizar consultas baseadas em condições aplicadas a elementos dentro de matrizes.

$all

O operador $all verifica se todos os elementos especificados existem em uma matriz.

db.collection.find({tags: {$all: ["red", "blue"]}});

$elemMatch

O operador $elemMatch verifica se ao menos um elemento em uma matriz satisfaz uma condição especificada.

db.collection.find({scores: {$elemMatch: { score: {$gt: 80}, grade: "A" }});

$size

O operador $size verifica se uma matriz tem um tamanho específico.

db.collection.find({tags: {$size: 3}});
