find()

O método find serve para consultar uma coleção com alguns critérios definidos, retornando uma lista com todos os documentos que se enquadram nos critérios. Utilizamos o código db.collection.find().

db.collection.find({nome: "José"}, {_id: 0, nome: 1, sobrenome: 1});
resultado: [ { nome: "José", sobrenome: "Aldo" }, { nome: "José", sobrenome: "Dias" } ]

O método find pode receber até dois parametros:

A query deve ser o primeiro parametro, ele será o filtro da sua consulta. Ele pode ser um documento simples, como no exemplo acima, para pegar registros com a mesma correspondencia nos campos especificados, mas também pode receber operadores lógicos, de comparação, de elementos, de avaliação e de matrizes. Deve ser informada entre chaves.

O projection é o segundo parâmetro, ele serve pra omitir campos desnecessarios ou expor apenas os campos desejados. O valor 0 omite um campo e o valor 1 expõe. Assim como a query ela deve ser informada entre chaves.

Para pegar todos os documentos da collection, basta passar esses dois parametros em chaves vazias, ou simplesmente não passa nenhum parametro.
